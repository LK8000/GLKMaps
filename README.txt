=========
 GLKMaps
=========

http://github.com/mpusz/GLKMaps

1. Description
==============
GLKMaps is a WWW download interface for LK8000 (http://www.lk8000.it) maps.
It is based on Google Maps framework. It parses LKMAPS templates and puts
them on the world map.

2. Project state
================
The project is still in Alpha phase so use it on your own responsibility.
Please report all the problems you encounter on the project homepage.
